# Sailfish-Pinephone-kernel-headers

Kernel headers are file that allow to create kernel driver. Here is the file for the Pinephone version of Sailfish 

## Install

For installing this headers you will just have to execute the install.sh script.  

```

bash install.sh

```

## Why this projects ?

I have seen that the kernel header are not available in the jolla reposetory.  
I did not find it and I needed it for a project, so I compile the kernel you get it.
